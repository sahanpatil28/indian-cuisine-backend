# Indian Cuisine Backend

## Description
This is the backend API for the Indian Cuisine project, built using **Node.js** and **Express**. The backend serves data related to Indian dishes, such as ingredients, preparation time, cooking time, and more. The API supports fetching all dishes, dish details, and suggesting dishes based on ingredients.

## Features
- **GET `/dishes`**: Returns a list of all dishes.
- **GET `/dishes/:name`**: Returns details of a specific dish by name.
- **GET `/dishes/suggest`**: Suggests dishes based on selected ingredients.
- **GET `/dishes/search`**: Searches dishes by name, ingredients, or origin.

## Technologies Used
- **Node.js**: JavaScript runtime for building the backend.
- **Express**: Web framework for building the API.
- **Cors**: For enabling cross-origin requests.
- **dotenv**: For loading environment variables.

## Requirements
- **Node.js** (version 14 or higher)
- **npm** (version 6 or higher)
